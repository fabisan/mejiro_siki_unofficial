## 直したいところ
- 2Uキーの幅はもう少し長いと見た目がきれい
- ステムの穴はわずかに大きい。
    応急処置方法:穴をふさぐようにマステを貼って、スイッチに押し付けて取り付ける。(ChatGPT先生直伝)
- スタビライザーを付けたい。Y,#,*がわずかにぐらつく
- nキーは、t,kの逆側中央を少し盛り上げたい。完全に平面だと押しづらい
- Pro Micro版も作りたい -> 結局RP2040になりました
